# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/3e5b24bd-f22d-4a29-8b34-af8ce9067845

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY`(https://aistudio.google.com/app/api-keys) in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
